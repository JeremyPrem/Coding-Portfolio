App Description: A user is presented with 2 options on the front page, the first is to enter their name and top three
favorite superheroes, Duplicate users will not be add, but their lists will be updated. Their data is added to a database.
The second option will use the chart.js API to retrieve everyone's top three heroes and display a bar graph
where each super is represented on the X axis and how many times they are in the database is on the Y axis

APIs used: https://www.chartjs.org/docs/latest/ and https://rapidapi.com/jakash1997/api/superhero-search/

Youtube demonstration: https://www.youtube.com/watch?v=o-YBDTqX_ZU
